Car Dealership Full Stack Capstone Project

---

## 📖 Project Description

This project is a full-stack web application developed as part of a capstone assignment. The application simulates a **Car Dealership system**, where users can:

* View dealership branches
* Browse available products (cars)
* View dealer pricing
* Submit and view reviews

The project follows a **microservices architecture** and integrates both frontend and backend services.

* Price comparison across dealers
* Review submission system
* Dynamic API integration
* Autoscaling using Kubernetes (HPA)
